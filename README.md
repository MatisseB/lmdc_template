# Le Monde de Camille - Publii Theme

Version: 1.0.0

A modern, responsive Publii theme inspired by the original Lemondedecamille website.

## About

This theme is a recreation of the Lemondedecamille WordPress website, now built as a Publii template. It features:

- Clean, modern design
- Responsive layout
- Tag and author pages
- Search functionality
- Customizable options

## Installation

1. Download the theme files
2. In Publii, go to Tools -> Themes
3. Click "Add theme" and select this theme folder
4. Activate the theme

## Customization

The theme can be customized through Publii's theme settings, including:

- Logo upload
- Color schemes
- Typography
- Layout options
