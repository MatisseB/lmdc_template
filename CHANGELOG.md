# Changelog

All notable changes to the Le Monde de Camille Publii theme will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-12

### Added
- Initial release of Le Monde de Camille theme
- Modern, responsive design based on Publii Blank theme
- Full support for Publii features including:
  - Block Editor
  - Tag pages with images
  - Author pages with images
  - Search functionality
  - Custom error pages (404)
  - Custom comments and embeds
  - Multiple menu positions
  - Related posts
  - Responsive images
- Grid-based layout with customizable widths
- Sidebar with configurable sections:
  - Selected posts
  - Featured posts
  - Authors list
- Comprehensive theme settings:
  - Layout customization (main and sidebar widths)
  - Font size adjustment
  - Logo upload
  - Favicon upload
- Clean, modern CSS with CSS variables for easy customization
- Fully responsive design (mobile, tablet, desktop)
- Modern typography with system font stack
- Navigation with hover effects and active states
- Styled pagination
- Enhanced post and page templates
- Author bio sections
- Featured image support
- Tag and author archive pages
- Comprehensive documentation
- Installation guide
- MIT License

### Design Features
- Primary color: #2c3e50 (Dark blue)
- Secondary color: #e74c3c (Red accent)
- Light background: #f8f9fa
- Border color: #e1e8ed
- System font stack for optimal performance
- Smooth transitions and hover effects
- Rounded corners and modern spacing
- Mobile-first responsive breakpoints

### Performance
- Minimal CSS (no bloat)
- System fonts (no external font loading)
- Lazy loading for images
- Optimized asset loading
- Fast rendering with CSS Grid

### Documentation
- README.md with quick start guide
- INSTALL.md with detailed installation steps
- DOCUMENTATION.md with complete feature documentation
- Inline code comments for developers
